# Interactive Tools Expansion

The team currently relies heavily on the `SimulationPlayground.tsx` and PathPlanner. If the software team requires new custom utilities (e.g., custom telemetry dashboard viewers, scouting data visualizers, or specific kinematics calculators), we should build them as interactive widgets embedded in the ARESWEB platform to keep everything unified.
