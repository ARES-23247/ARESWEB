# Research: Features

## Table Stakes
- **Album Creation**: Create, rename, delete albums.
- **Media Mapping**: Associate uploaded/imported R2 media with an album.
- **Masonry Layout**: Core grid view for displaying varying aspect ratio photos efficiently.

## Differentiators
- **Moving/Marquee Gallery**: A dynamic, scrolling layout for engaging public displays.
- **Google Photos Ingestion**: Bulk ingest photos into an album via the existing Google Picker.

## Complexity Notes
- Handling aspect ratios and responsive columns for Masonry can be tricky across viewport sizes.
