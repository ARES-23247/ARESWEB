# Curate Initial Experiments

Before launching the Science Corner, we should build 2-3 fun, non-robotics visual toys or experiments to seed the library.

**Ideas:**
- Conway's Game of Life (great for showing grid rendering)
- 2D Gravity Simulator (great for basic physics math)
- Fractal Generator (Mandelbrot set or L-systems)

These should be built as `.tsx` templates that can load directly into the new Sandbox infrastructure.
