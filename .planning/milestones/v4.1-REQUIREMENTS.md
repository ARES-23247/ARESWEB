# Milestone v4.1 Requirements

## Media Integration & API Modernization

This milestone retroactively documents the significant media integration work and API migrations completed after v4.0.

### Google Photos Integration
- [ ] **PHOTOS-01**: Migrate from deprecated Google Photos Library API to Picker API v1 for file selection
- [ ] **PHOTOS-02**: Implement JIT (Just-In-Time) album sync UI for task attachments
- [ ] **PHOTOS-03**: Add Google Photos extension to Tiptap editor for rich media embedding
- [ ] **PHOTOS-04**: Build YouTube video upload flow with automatic Google Photos album integration

### YouTube Integration
- [ ] **YOUTUBE-01**: Implement YouTube video picker modal for selecting and embedding videos
- [ ] **YOUTUBE-02**: Add YouTube upload API integration with progress tracking
- [ ] **YOUTUBE-03**: Create Tiptap extension for YouTube embeds with preview thumbnails

### Google Drive Integration
- [ ] **DRIVE-01**: Integrate Google Drive Picker API v3 for task document attachments
- [ ] **DRIVE-02**: Add document creation shortcuts from Drive picker and dashboard
- [ ] **DRIVE-03**: Implement loading states for async document creation/deletion operations

### TBA API Migration
- [ ] **TBA-01**: Replace deprecated TBA (The Blue Alliance) API with ftc-events package
- [ ] **TBA-02**: Update event synchronization logic to use new ftc-events schema
- [ ] **TBA-03**: Migrate all TBA API tests to ftc-events test suite

### Bug Fixes & Polish
- [ ] **FIX-01**: Resolve task label assignment 500 error due to foreign key constraints
- [ ] **FIX-02**: Fix iframe blocking for embedded documents by enforcing /preview URL replacement
- [ ] **FIX-03**: Resolve Tiptap v3 collaboration cursor DOM rendering collision
- [ ] **FIX-04**: Fix ESLint/TypeScript errors across google-photos and events routes

### CI/CD Improvements
- [ ] **CI-01**: Upgrade CI from pnpm v9 to v11 to match local lockfile format
- [ ] **CI-02**: Add pnpm build script approvals to prevent ERR_PNPM_IGNORED_BUILDS errors
- [ ] **CI-03**: Configure pnpm exec for Playwright to fix module resolution
