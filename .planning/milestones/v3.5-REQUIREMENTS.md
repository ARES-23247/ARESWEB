# Requirements Archive: v3.5

## Version Control
- [x] **VER-01**: User can view a chronological list of historical snapshots for a document, post, or event.
- [x] **VER-02**: User can preview the rich-text content of a selected historical snapshot.
- [x] **VER-03**: User can restore a historical snapshot, which safely overwrites the current `content_draft` without immediately publishing to the live production state.

## Contributor Attribution
- [x] **CON-01**: System accurately tracks all unique users who contribute to a document's history.
- [x] **CON-02**: System displays an overlapping stack of contributor avatars near the title on public-facing pages (blog posts, events, docs).
- [x] **CON-03**: System automatically filters out non-student roles (mentors, coaches, adults) from the public contributor list to comply with FIRST Youth Protection/COPPA guidelines.
