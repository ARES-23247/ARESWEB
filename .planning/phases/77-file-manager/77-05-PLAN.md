2. Import UploadedFile type from "../../shared/routes/files"

3. Add state for modal:
   - const [isFileBrowserOpen, setIsFileBrowserOpen] = useState(false)

4. Add handler for file selection:
   ```typescript
   const handleFileSelect = (file: UploadedFile) => {
     if (!editor) return;
     const linkText = file.title || file.filename;
     const markdown = `[${linkText}](/api/files/download/${file.id})`;
     editor.commands.insertContent(markdown);
     setIsFileBrowserOpen(false);
   };
   ```

5. Pass onInsertFileLink prop to RichEditorToolbar:
   - onInsertFileLink={() => setIsFileBrowserOpen(true)}

6. Render FileBrowserModal after AssetPickerModal:
   ```tsx
   <FileBrowserModal
     isOpen={isFileBrowserOpen}
     onClose={() => setIsFileBrowserOpen(false)}
     onSelect={handleFileSelect}
   />
   ```

7. The markdown link format per D-15:
   - Uses title if available, else filename
   - URL: /api/files/download/{id}
   - Inserts at current cursor position

DO NOT use @ts-ignore or any type assertions.
