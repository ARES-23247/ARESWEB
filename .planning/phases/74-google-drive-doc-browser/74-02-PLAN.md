Replace the placeholder /files route in `functions/api/routes/google-drive/index.ts`:

1. Import listDriveFilesRoute from shared/routes/google-drive.ts
2. Replace placeholder route with openapi(listDriveFilesRoute, async (c) => {...})
3. Extract query params: q, pageToken, pageSize from c.req.valid("query")
4. Get access token: const token = await getDriveAccessToken(getDb(c), c.env)
5. Build Drive API query:
   - Base URL: https://www.googleapis.com/drive/v3/files
   - q parameter combines:
     * Folder filter: "'{GOOGLE_DRIVE_FOLDER_ID}' in parents"
     * Trashed filter: "and trashed=false"
     * MIME type filter per D-03: "and mimeType in ('application/vnd.google-apps.document', 'application/vnd.google-apps.spreadsheet', 'application/vnd.google-apps.presentation', 'application/vnd.google-apps.drawing')"
     * Name search if q provided: "and name contains '{escaped_q}'"
   - Fields: "nextPageToken,files(id,name,mimeType,modifiedTime,webViewLink,owners)"
   - pageSize: default 50 per D-12
   - pageToken: if provided
6. Fetch with Authorization: Bearer {token}
7. Transform response: map owners[0].displayName to owner field
8. Return c.json(transformed, 200)
9. Throw ApiError on non-200 response

DO NOT add D1 caching per D-13 (deferred to Phase 77).
</action>
