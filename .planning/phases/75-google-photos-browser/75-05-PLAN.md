## Photos API Upload Documentation

- mediaItems:batchCreate endpoint: POST https://photoslibrary.googleapis.com/v1/mediaItems/batchCreate
- Request body:
  ```json
  {
    "albumId": "optional-album-id",
    "newMediaItems": [
      {
        "description": "optional description",
        "simpleMediaItem": {
          "uploadToken": "token-from-upload-call",
          "fileName": "photo.jpg"
        }
      }
    ]
  }
  ```
- Upload flow:
  1. POST to https://photoslibrary.googleapis.com/v1/uploads with binary file data
  2. Response contains upload token (raw string, not JSON)
  3. Use upload token in batchCreate call
