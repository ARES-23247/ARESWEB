## Photos API Documentation

- mediaItems:search endpoint: POST https://photoslibrary.googleapis.com/v1/mediaItems:search
- Request body: { pageSize: number, pageToken?: string, albumId?: string }
- Response: { mediaItems: [...], nextPageToken?: string }
- For unfiltered listing, use empty body {} which returns all media items

## Key Decisions from CONTEXT.md

- **D-01:** Filter server-side by mimeType — exclude video/mp4, video/quicktime, video/x-msvideo
- **D-06:** Page size 25 with pageToken pagination
- **D-10:** Click album to filter media items by albumId
