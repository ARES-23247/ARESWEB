    1. **GET /api/onshape/bom/:documentId/:elementId**
       - Requires cf-access-authenticated-user-email (Zero Trust)
       - Fetches BOM from Onshape: GET /api/assemblies/{eid}/bom
       - Returns hierarchical or flat BOM structure
       - Response structure:
         ```typescript
         interface BOMResponse {
           documentId: string;
           elementId: string;
           elementName: string;
           parts: BOMPart[];
           totalParts: number;
           lastSyncedAt: string;
         }

         interface BOMPart {
           partId: string;
           partName: string;
           partNumber: string;
           quantity: number;
           material: string;
           mass: number; // in grams
           configuration: string;
         }
         ```
       - Records sync in onshape_bom_history table

    2. **GET /api/onshape/bom/history/:documentId**
       - Requires cf-access-authenticated-user-email
       - Returns sync history from D1
       - Response: { history: BOMHistoryEntry[] }
       - Ordered by most recent first

    3. **GET /api/onshape/bom/export/:documentId/:elementId**
       - Requires cf-access-authenticated-user-email
       - Returns BOM as CSV file
       - Headers: Content-Type: text/csv, Content-Disposition
       - Columns: Part Name, Part Number, Quantity, Material, Mass (g)

    Update main router:
    ```typescript
    // functions/api/routes/onshape/index.ts
    import bom from './bom';
    app.route('/bom', bom);
    ```

    Implementation details:
    ```typescript
    app.get('/bom/:documentId/:elementId', typedHandler({
      params: z.object({
        documentId: z.string().length(24),
        elementId: z.string().length(24),
      }),
    }, async (c, { params }) => {
      const userId = c.get('cf-access-authenticated-user-email');
      const token = await getOnshapeToken(userId);

      // Fetch from Onshape
      const response = await fetchOnshape<BOMData>(
        userId,
        `/assemblies/${params.elementId}/bom`
      );

      // Record sync history
      await env.DB.insert(onshapeBomHistory).values({
        documentId: params.documentId,
        elementId: params.elementId,
        partCount: response.data.parts.length,
        syncedBy: userId,
        syncedAt: new Date(),
      });

      return c.json({
        documentId: params.documentId,
        elementId: params.elementId,
        parts: response.data.parts,
        totalParts: response.data.parts.length,
        lastSyncedAt: new Date().toISOString(),
      });
    }));
    ```
