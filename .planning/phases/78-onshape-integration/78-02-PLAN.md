       - Generates thumbnail URL for document/element

       - Fetches single document details from Onshape
       - Caches if public
       - Returns: Document with elements array

    4. **GET /api/onshape/documents/:documentId/elements**
       - Requires cf-access-authenticated-user-email
       - Fetches all elements in document
       - Returns: { elements: Element[] }
