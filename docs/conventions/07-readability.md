# Documentation Readability

> Enforces 8th-grade reading level (Flesch-Kincaid < 8.0) for all user-facing text.

## Rule: 8th Grade Reading Level

Complex concepts must be explained simply. Follow this framework:

## 1. Active Voice Only

- ❌ "The website is hydrated by React."
- ✅ "React hydrates the website."

## 2. De-Jargonification

- ❌ "Utilize the interface to ascertain telemetry vectors."
- ✅ "Use the dashboard to find the robot's data."

## 3. Sentence Length

- Keep under 15-20 words when possible
- Break anything >25 words into multiple statements

## 4. Simple Analogies

Example: "A Cloudflare Edge Worker is like a drive-thru window serving the website right next to the user."

## 5. "In Other Words" Sections

After introducing technical terms, append a plain-language rephrasing.

## 6. Concrete First, Abstract Second

Show what a feature *does* before explaining *how* it works.
