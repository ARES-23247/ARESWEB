# ARES Institutional Standards

## 1. Technical Excellence (The 12 Pillars)
1.  **Security**: Auth middleware, bound D1 params, Zod validation.
2.  **Privacy**: COPPA/YPP (no PII leak), AES for sensitive data.
3.  **Accessibility**: WCAG 2.1 AA, semantic HTML, keyboard nav.
4.  **Brand**: Palette (`ares-red`/`gold`), `League Spartan` font.
5.  **Efficiency**: Query JOINs/batching, Zustand state, modular imports.
6.  **Refactoring**: Component modularization, DRY enforcement.
7.  **Portability**: Relative routing, dynamic `env` usage.
8.  **Functionality**: Schema sync, soft-delete, failure exposure.
9.  **Testing**: Playwright E2E core flows, 85%+ Vitest coverage.
10. **Architecture**: Middleware sequencing, router mounting.
11. **Hygiene**: No scratch files, no `console.log`, clean tree.
12. **Resilience**: `c.executionCtx.waitUntil()` for side effects.

## 2. Cultural Legacy (FIRST Values)
- **Values**: Discovery, Innovation, Impact, Inclusion, Teamwork, Fun.
- **Ethos**: "Robots are the vehicle; students are the cargo."
- **Mindset**: "Mountaineer Mindset" (Grit/Determination).
- **Awards**: Proactively document journey for **Inspire**, **Think**, and **Connect** awards.
